## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby: 2.7.1
Rails: 5.2.4
Postgres: >=9.5

### Setup

```
$ cd <folder>
$ git clone git@github.com:ldelbel/ror-social-scaffold.git
$ cd ror-social-scaffold
```

Instal gems with:

```
bundle install
```

Setup database with:

```
$ rails db:migrate
```

### Usage

Start server with:

```
$ rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
 $ rspec
```
